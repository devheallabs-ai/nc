**Note on Website Files:**
The HTML, CSS, and JS files for the public documentation site are NOT tracked in this main repository branch to keep standard `git clone` sizes as small as possible for developers.

If you are a maintainer looking for the website files (`index.html`, `vow.css`, etc.), they are managed in the separate `gh-pages` branch or the `nc-html-website` repository.
