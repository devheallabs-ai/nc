# NC Community Code of Conduct (v1)

## Purpose

NC is an open-source project and a shared learning space. We want discussions, reviews, and collaboration to stay respectful, practical, and welcoming for contributors of all experience levels.

## Expected behavior

- Be respectful in issues, pull requests, discussions, and chat.
- Give technical feedback on ideas and code, not on people.
- Assume good intent, ask clarifying questions, and avoid hostile tone.
- Share credit, help unblock others, and keep conversations constructive.

## Unacceptable behavior

- Harassment, hate speech, personal attacks, intimidation, or threats.
- Doxxing, sharing private information, or repeated unwanted contact.
- Sexualized language or imagery in project spaces.
- Deliberate disruption, trolling, bad-faith arguments, or spam.

## Scope

This policy applies across NC community spaces, including:

- This repository and related repositories under the NC project.
- GitHub issues, pull requests, and discussions.
- Public and private communication channels used for project collaboration.

## Reporting

Report concerns to: `support@devheallabs.in`

When reporting, include any relevant context (links, screenshots, timestamps). Reports are handled as confidentially as practical.

## Maintainer responsibilities

Project maintainers are responsible for clarifying standards and taking fair, consistent action when behavior violates this policy.

## Enforcement

Depending on severity and pattern, actions may include:

- Warning and request to change behavior.
- Temporary moderation limits or removal from discussion spaces.
- Suspension or permanent ban from project participation.

Retaliation against anyone reporting a concern is not allowed.
