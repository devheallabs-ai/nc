# Changelog — NC Language (VSCode Extension)

All notable changes to the NC Language VSCode extension will be documented here.

## [1.0.0] — 2026-03-30

### Added
- Full syntax highlighting for `.nc` and `.ncui` files
- 35+ code snippets for common NC patterns
- Hover documentation for NC keywords and built-ins
- Run, Serve, and Validate commands from Command Palette
- Integrated REPL support
- Language Server Protocol (LSP) integration
- Auto-validate on save
- NC-specific file icons
