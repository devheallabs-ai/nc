# NeuralEdge

**AI-driven financial intelligence engine built with NC.**

NeuralEdge is a self-contained market analysis and prediction service powered by the NC programming language. It combines real-time data ingestion, 30+ technical indicators, and machine-learning predictions into a single deployable unit.

Website: [neuraledge.devheallabs.in](https://neuraledge.devheallabs.in)

---

## Features

- **30+ Technical Indicators** -- RSI, MACD, Bollinger Bands, Stochastic Oscillator, ATR, OBV, VWAP, Ichimoku Cloud, Fibonacci levels, and more.
- **Live Market Data** -- Real-time price feeds via configurable data providers.
- **ML Predictions** -- Ensemble models (LSTM, gradient boosting, transformer) with confidence scoring.
- **Sentiment Analysis** -- News and social-media sentiment aggregation through AI.
- **Pattern Recognition** -- Automated detection of chart patterns (head-and-shoulders, double tops/bottoms, wedges).
- **Portfolio Optimization** -- Risk-adjusted allocation suggestions using Modern Portfolio Theory.
- **Anomaly Detection** -- Volume and price anomaly alerts.
- **Backtesting Engine** -- Historical strategy validation with detailed performance metrics.
- **Multi-Asset Support** -- Equities, crypto, forex, and commodities.
- **REST API** -- Clean JSON API for integration with dashboards, bots, and third-party tools.
- **Local AI Support** -- Optional Ollama sidecar for fully offline AI inference.
- **Containerized Deployment** -- Docker and Kubernetes manifests included.

## Quick Start

### Prerequisites

- NC compiler/runtime (v0.1+)
- API key from a supported market data provider (see `.env.example`)

### Setup

```bash
# Clone the repository (if standalone)
git clone https://github.com/DevHealLabs/neuraledge.git
cd neuraledge

# Run setup to create data directories and config
./start.sh setup

# Edit configuration
cp .env.example .env
# Fill in your API keys in .env
```

### Run

```bash
./start.sh
```

The server starts on `http://localhost:8000` by default.

## Configuration Reference

All configuration is via environment variables. See `.env.example` for the full list.

| Variable | Default | Description |
|----------|---------|-------------|
| `NEURALEDGE_PORT` | `8000` | Server listen port |
| `NEURALEDGE_HOST` | `0.0.0.0` | Server bind address |
| `NEURALEDGE_DATA_DIR` | `./data` | Directory for cache and model data |
| `NEURALEDGE_LOG_LEVEL` | `info` | Log verbosity (debug, info, warn, error) |
| `MARKET_DATA_PROVIDER` | `yahoo` | Data provider (yahoo, alpha_vantage, polygon) |
| `MARKET_DATA_API_KEY` | -- | API key for market data provider |
| `OLLAMA_HOST` | `http://localhost:11434` | Ollama endpoint for local AI |
| `OLLAMA_MODEL` | `llama3` | Model name for Ollama inference |
| `PREDICTION_HORIZON_DAYS` | `7` | Default prediction lookahead |
| `CACHE_TTL_SECONDS` | `300` | Data cache time-to-live |
| `MAX_CONCURRENT_REQUESTS` | `50` | Request concurrency limit |

// NeuralEdge — AI-driven financial intelligence engine
// Main service entry point
//
// This is the service skeleton. Each endpoint handler delegates to
// modules in the full implementation (indicators, predictions, etc.).

    port: env("NEURALEDGE_PORT", "8000"),
    host: env("NEURALEDGE_HOST", "0.0.0.0")
}

// -- Configuration --
let config = {
    data_dir:   env("NEURALEDGE_DATA_DIR", "./data"),
    log_level:  env("NEURALEDGE_LOG_LEVEL", "info"),
    provider:   env("MARKET_DATA_PROVIDER", "yahoo"),
    api_key:    env("MARKET_DATA_API_KEY", ""),
    cache_ttl:  int(env("CACHE_TTL_SECONDS", "300")),
    ollama:     env("OLLAMA_HOST", "http://localhost:11434"),
    model:      env("OLLAMA_MODEL", "llama3"),
    horizon:    int(env("PREDICTION_HORIZON_DAYS", "7"))
}

// NeuralEdge — Test Suite
// Run with: nc test tests/test_neuraledge.nc

test "health endpoint returns 200" {
    let res = http_get("http://localhost:8000/health")
    assert_eq(res.status, 200)
    assert_eq(res.json().status, "ok")
    assert_eq(res.json().service, "NeuralEdge")
}

test "about endpoint returns version info" {
    let res = http_get("http://localhost:8000/about")
    assert_eq(res.status, 200)
    assert_eq(res.json().name, "NeuralEdge")
    assert(res.json().version != "", "version should not be empty")
    assert(res.json().indicators >= 30, "should report 30+ indicators")
}

test "configuration loads defaults" {
    // Verify environment-based config falls back to defaults
    let port = env("NEURALEDGE_PORT", "8000")
    assert_eq(port, "8000")

    let provider = env("MARKET_DATA_PROVIDER", "yahoo")
    assert_eq(provider, "yahoo")

    let log_level = env("NEURALEDGE_LOG_LEVEL", "info")
    assert_eq(log_level, "info")
}

test "data directory can be created" {
    let data_dir = env("NEURALEDGE_DATA_DIR", "./data")
    mkdir_p(data_dir + "/cache")
    mkdir_p(data_dir + "/models")
    mkdir_p(data_dir + "/logs")

    assert(dir_exists(data_dir + "/cache"), "cache dir should exist")
    assert(dir_exists(data_dir + "/models"), "models dir should exist")
    assert(dir_exists(data_dir + "/logs"), "logs dir should exist")
}

test "quote endpoint responds" {
    let res = http_get("http://localhost:8000/api/v1/quote/AAPL")
    assert_eq(res.status, 200)
    assert_eq(res.json().symbol, "AAPL")
